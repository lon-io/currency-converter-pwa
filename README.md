# pwa-currency-converter
---
Currency Converter App
(Built as a Project for the ALC Udacity Program)

Features:
---
- Offline First
- IndexDB
- Cache API
- Service Worker
- DOM API
- Mobile-first Responsive UI

Usage
---

Start the development server with this command:

```
npm run start
```

Setup
---

```
npm install
```

Compile
---

```
npm run build
```

Deploy (Github Pages)
---

```
npm run deploy
```
